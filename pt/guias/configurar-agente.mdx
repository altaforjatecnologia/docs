## Objetivo

Defina o que o agente deve fazer:

```text
Exemplo: Atender clientes interessados em consultas 
odontológicas, fornecer informações sobre tratamentos 
e agendar avaliações.
```

<Accordion title="Exemplos por tipo de negócio">
**E-commerce:**
```
Atender clientes, apresentar produtos do catálogo,
responder dúvidas sobre pedidos e fornecer informações
sobre entrega.
```

**Serviços:**
```
Fornecer informações sobre serviços, enviar orçamentos
e agendar reuniões com clientes interessados.
```

**Imobiliária:**
```
Atender interessados em imóveis, fornecer informações
sobre disponibilidade e agendar visitas.
```
</Accordion>

## Teste

Clique no ícone de chat no canto inferior direito para testar o agente antes de colocá-lo no ar.

<Steps>
  <Step title="Abrir Chat">
    Clique no ícone de chat
  </Step>
  
  <Step title="Fazer Perguntas">
    Simule conversas como se fosse um cliente
  </Step>
  
  <Step title="Ajustar">
    Se necessário, volte e ajuste o objetivo
  </Step>
</Steps>
