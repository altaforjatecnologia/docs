## Acessar

No menu, clique em **Flows** para ver formulários existentes ou criar novos.

## Criar Flow

<Steps>
  <Step title="Novo Flow">
    Clique em **Novo Flow**
  </Step>
  
  <Step title="Nome e Descrição">
    - Nome: Como o Flow aparece na lista
    - Descrição: Para que serve (apenas para você)
  </Step>
  
  <Step title="Adicionar Campos">
    Arraste componentes da barra lateral para o formulário
  </Step>
  
  <Step title="Configurar Campos">
    Clique em cada campo para configurar:
    - Label (texto que o cliente vê)
    - Placeholder (exemplo)
    - Obrigatório (sim/não)
  </Step>
  
  <Step title="Testar">
    Use o botão **Preview** para ver como fica no WhatsApp
  </Step>
  
  <Step title="Publicar">
    Clique em **Publicar** para ativar
  </Step>
</Steps>

## Tipos de Campo

### Texto
- **Texto curto:** Nome, email, cidade
- **Texto longo:** Descrição, observações
- **Email:** Com validação automática
- **Telefone:** Com formatação automática
- **Número:** Apenas números

### Seleção
- **Múltipla escolha:** Cliente escolhe UMA opção
- **Checkboxes:** Cliente escolhe VÁRIAS opções
- **Dropdown:** Lista suspensa (quando tem muitas opções)

### Data e Hora
- **Data:** Calendário para selecionar
- **Hora:** Seletor de horário

### Outros
- **Consentimento:** Checkbox de aceite (LGPD)
- **Imagem:** Upload de foto
- **Documento:** Upload de arquivo

## Boas Práticas

<Check>
  Use labels claros e diretos
</Check>
<Check>
  Marque apenas campos essenciais como obrigatórios
</Check>
<Check>
  Adicione placeholders com exemplos
</Check>
<Check>
  Teste no celular antes de publicar
</Check>
<Check>
  Mantenha formulários curtos (máximo 7-8 campos)
</Check>

<Warning>
  Formulários muito longos têm alta taxa de abandono. Se precisa de muitas informações, divida em 2 Flows ou use conversa + Flow.
</Warning>

## Como o Agente Usa

Depois de publicado, configure quando o agente deve enviar o Flow:

1. Vá em **Meu Agente**
2. Nas instruções, adicione:

```text
Quando o cliente demonstrar interesse em agendar, 
envie o Flow "Agendamento de Consulta" para capturar 
as informações necessárias.
```

O agente identifica o momento certo na conversa e envia automaticamente.

## Dados Capturados

Após o cliente enviar o Flow:
- Dados ficam salvos no perfil do lead
- Você vê em **Conversas** > detalhes do lead
- Pode acionar automações com base nos dados
