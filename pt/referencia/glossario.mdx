## A

**Agente de IA**
Assistente virtual inteligente que conversa automaticamente com clientes via WhatsApp, utilizando Inteligência Artificial.

**API (Application Programming Interface)**
Interface que permite comunicação entre diferentes sistemas de software.

**Autenticação**
Processo de verificar a identidade de um usuário ou sistema.

## B

**Base de Conhecimento**
Conjunto de documentos, textos e informações que o agente de IA utiliza para responder perguntas.

**Batch Processing**
Processamento de múltiplas operações em lote, ao invés de uma por vez.

**Bot**
Programa automatizado que executa tarefas repetitivas. No contexto do Integrabot, refere-se ao agente de IA.

## C

**Canal**
Conexão do WhatsApp Business vinculada ao Integrabot. Você pode ter múltiplos canais.

**Chunking**
Divisão de documentos grandes em pedaços menores para melhor processamento.

**CRM (Customer Relationship Management)**
Sistema de gestão de relacionamento com clientes.

**Conversa**
Troca de mensagens entre um cliente e o agente (ou atendente humano).

## D

**Dashboard**
Painel de controle com métricas e visão geral do sistema.

**Deploy**
Processo de colocar uma aplicação ou atualização em produção.

## E

**Embedding**
Representação numérica (vetor) de um texto que captura seu significado semântico. Também chamado de vetor.

**Endpoint**
URL específica de uma API que executa uma função.

**Evolution API**
API alternativa para integração com WhatsApp, usada como opção no Integrabot.

## F

**Ferramenta de IA**
Função que o agente pode executar, como consultar estoque ou criar pedido.

**Flow / WhatsApp Flow**
Formulário interativo nativo do WhatsApp para capturar informações estruturadas.

## G

**GPT (Generative Pre-trained Transformer)**
Modelo de linguagem desenvolvido pela OpenAI, usado no Integrabot.

## H

**Handoff**
Transferência de uma conversa do agente de IA para atendimento humano.

**Headers HTTP**
Metadados enviados em requisições HTTP, como autenticação e tipo de conteúdo.

## I

**Integração**
Conexão entre o Integrabot e outro sistema (CRM, calendário, etc).

**Intervenção Manual**
Quando um atendente humano assume uma conversa que estava sendo atendida pela IA.

## J

**JSON (JavaScript Object Notation)**
Formato de dados estruturado usado para comunicação entre sistemas.

**JSON Schema**
Especificação para definir estrutura e validação de dados JSON.

## K

**KPI (Key Performance Indicator)**
Indicador-chave de desempenho, métrica importante para o negócio.

**Knowledge Base**
Ver Base de Conhecimento.

## L

**Lead**
Potencial cliente que demonstrou interesse em seus produtos ou serviços.

**LLM (Large Language Model)**
Modelo de linguagem grande, como GPT-4 ou Claude, que gera texto natural.

**LGPD**
Lei Geral de Proteção de Dados, regulamentação brasileira sobre privacidade.

## M

**Meta**
Empresa (anteriormente Facebook) que desenvolve o WhatsApp.

**Metadados**
Dados sobre dados. Informações adicionais sobre documentos, como categoria e data.

**Middleware**
Camada intermediária de software que conecta diferentes sistemas.

## N

**NLP (Natural Language Processing)**
Processamento de Linguagem Natural, tecnologia que permite computadores entenderem texto humano.

**Notificação**
Alerta enviado quando ocorre um evento importante (nova mensagem, lead, etc).

## O

**OAuth**
Protocolo de autorização usado para conectar aplicações de forma segura.

**OpenAI**
Empresa que desenvolve modelos de IA como GPT-4.

**Opt-in**
Consentimento explícito do usuário para receber comunicações.

## P

**Pipeline**
Sequência de etapas de processamento de dados.

**Prompt**
Instrução ou contexto fornecido a um modelo de IA para gerar resposta.

**Prompt Engineering**
Arte de escrever instruções eficazes para modelos de IA.

## Q

**QR Code**
Código de barras bidimensional usado para conectar WhatsApp.

**Query**
Consulta ou pergunta feita a um banco de dados ou sistema de busca.

**Queue (Fila)**
Estrutura de dados onde tarefas aguardam processamento em ordem.

## R

**RAG (Retrieval Augmented Generation)**
Técnica que combina busca de informação com geração de texto pela IA.

**Rate Limit**
Limite de requisições por período de tempo para evitar sobrecarga.

**Reranking**
Reordenação de resultados de busca para melhorar relevância.

**Retry**
Tentativa de reexecutar uma operação que falhou.

## S

**Sentiment Analysis**
Análise de sentimento, identificação se texto é positivo, negativo ou neutro.

**Serverless**
Arquitetura onde código roda sob demanda sem gerenciar servidores.

**Similaridade Semântica**
Medida de quão similares dois textos são em significado, não necessariamente em palavras.

**Slug**
Versão simplificada de um texto para uso em URLs (ex: "minha-empresa").

**Submissão**
Resposta completa de um formulário (Flow) preenchido pelo cliente.

## T

**Tag**
Etiqueta para classificar e organizar leads, conversas ou documentos.

**Temperatura**
Parâmetro que controla criatividade vs consistência nas respostas da IA (0.1-1.0).

**Token**
1. Unidade de texto processada por modelos de IA (~4 caracteres)
2. Chave de autenticação para APIs

**Transferência**
Ver Handoff.

## U

**URL (Uniform Resource Locator)**
Endereço web (ex: https://integrabot.ai).

**UX (User Experience)**
Experiência do usuário, como ele interage com o sistema.

## V

**Vetor / Vector**
Ver Embedding.

**Vetorização**
Processo de converter texto em vetores (embeddings).

**Vector Database**
Banco de dados especializado em armazenar e buscar vetores.

## W

**Webhook**
Notificação automática enviada por um sistema quando ocorre um evento.

**WhatsApp Business API**
Interface oficial do WhatsApp para empresas automatizarem comunicação.

**Workspace**
Espaço de trabalho isolado no Integrabot. Cada empresa/projeto tem seu workspace.

## X

**XSS (Cross-Site Scripting)**
Tipo de vulnerabilidade de segurança em aplicações web.

## Z

**Zapier**
Plataforma de automação que conecta diferentes aplicações.
